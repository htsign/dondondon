# dondondon

## これは何

自分が使うために作ったMastodonクライアントです

![image](https://user-images.githubusercontent.com/1310799/74102142-2bc02380-4b84-11ea-9060-c6fd82b37ae8.png)


## できること
* ホームタイムラインの閲覧
* 取得したホームタイムラインの検索
* ユーザー個別のタイムライン閲覧
* ふぁぼ・返信
* 通知の閲覧

## できないこと

* ショートカットキーとかろくに実装してません
* 返信など他のタイムラインの閲覧
* 複垢には対応してません
* Stream(リアルタイムにTLに反映されるやつ)はまだ実装していません
* リブログ（ブースト）できません

## この先実装したい機能
* 抽出タブ
* カスタム絵文字
* 画像投稿
* 複数枚の画像表示と拡大縮小
* 通知音
